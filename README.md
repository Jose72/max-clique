# max-clique

Distributed algorithms for NP-complete MCP - Maximum Clique Problem

### Run

```bash
 $ cd pattern-algorithm
 $ ./pattern-distr.py [--workers=NUM(#cpus) | --graph=PATH(null)]
```

- Clean

```bash
 $ make clean
```

### Tests

```bash
 $ make generate
```

- Clean
